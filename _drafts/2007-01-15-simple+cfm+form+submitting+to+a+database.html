--- layout: post title: Simple CFM form submitting to a database date: '2007-01-15T13:40:00.000-08:00' author: Bryan tags: modified_time: '2007-01-15T14:02:42.341-08:00' blogger_id: tag:blogger.com,1999:blog-19111454.post-235982025471044887 --- what you need:
  • Coldfusion Development Server (you can download a single user developer version here: http://www.adobe.com/products/coldfusion/trial/. The current version as of this writing is 7.0.2)
  • Microsoft Access 2000 or later
  • A plain text editor (notepad will do)
  1. install coldfusion on your box. we will use all the default install locations for this example.
  2. create the following folders in the webroot:
    1. database
    2. web
  3. create the master database to use in these examples:
    1. in the database directory create a new access database called "master.mdb"
    2. create a new table in design view
    3. add the following columns to the table:
      1. user_id (AutoNumber datatype, set as primary key)
      2. user_name (text datatype, field size of 20)
      3. user_password (text datatype, field size of 20)
      4. user_full_name (text_datatype, field size of 255)
    4. save the new table with a name of "users" and close the design view.
  4. add the newly created database as a coldfusion datasource. This will allow coldfusion to access the database we just created.
    1. login to the coldfusion admin page on your local machine using the admin username and password you set durring the coldfusion install (http://127.0.0.1/cfide/administrator)
